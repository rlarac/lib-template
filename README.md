# lib-template

library template with typescript &amp; rollup &amp; vite

## Install

```bash
npm install

# or

yarn install
```

## Dev

### Playgroud

```bash
npm run dev

# or

yarn dev
```

Then open [http://localhost:8080](http://localhost:8080) in browser

### Documentation

```bash
npm run docs:dev

# or

yarn docs:dev
```

Then open [http://localhost:3000](http://localhost:3000) in browser

## Test

```bash
npm run test

# or

yarn test
```

## Build

```bash
npm run build

# or

yarn build
```

## Build Documentation

```bash
npm run docs:build

# or

yarn docs:build
```
